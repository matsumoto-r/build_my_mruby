## build my mruby script

You can use mruby with some mrbgems easily by this script

* includ modules

        mruby-json
        mruby-uv
        mruby-http
        mruby-redis
        mruby-curl
        mruby-md5
        mruby-sha1
        mruby-base64
        mruby-simplehttp
        mruby-httprequest
        mruby-growthforecast
        mruby-zabbix
        mruby-random
        mruby-mtest


* install

        sh mymruby_with_mrbgems.sh
        cd mruby
        ./bin/mruby --version
